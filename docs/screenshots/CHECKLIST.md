# Screenshots Checklist

Use this checklist before publishing screenshots in `docs/screenshots/`.

- [ ] Capture: landing page (`landing-page.png`)
- [ ] Capture: user dashboard (`user-dashboard.png`)
- [ ] Capture: referrals page (`referrals-page.png`)
- [ ] Capture: admin dashboard (`admin-dashboard.png`)
- [ ] Use consistent width (recommended 1920px desktop)
- [ ] Remove sensitive data (emails, wallet addresses, API keys)
- [ ] Keep filenames exactly as referenced in `README.md`
- [ ] Commit images to `docs/screenshots/`
- [ ] Verify images render on GitHub README preview
